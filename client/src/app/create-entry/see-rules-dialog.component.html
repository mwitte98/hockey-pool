Rules

  • Entries must be submitted by Monday, May 2nd 7pm ET
  • After the entry submission deadline, if a player misses a game for any reason, they cannot be substituted
  • Entry fee: $20 per entry
  • Unlimited number of entries
  • Payment can be sent through Venmo or PayPal to Al Kurek
  • Payouts: 25% to 1st place, 15% to 2nd place, and remaining 60% will be determined based on the number of entries

  • Entries will consist of 16 players; 1 from each team
  • 16 players will be broken down into the following positions:
    • {{ settingsService.setting.minCenters }} centers
    • {{ settingsService.setting.minWingers }} wingers
    • {{ settingsService.setting.minDefensemen }} defensemen
    • {{ settingsService.setting.minGoalies }} goalies
    • 1 flex center OR winger OR defenseman

  • Scoring:
    • Goal: {{ settingsService.setting.pointsGoals }} points
    • Assist: {{ settingsService.setting.pointsAssists }} point
    • Game Winning Goal: {{ settingsService.setting.pointsGwg }} point
    • Overtime Goal: {{ settingsService.setting.pointsOtg }} points
    • Short Handed Goal: {{ settingsService.setting.pointsShg }} points
    • Goalie Win: {{ settingsService.setting.pointsWins }} points
    • Goalie Overtime Loss: {{ settingsService.setting.pointsOtl }} point
    • Goalie Shutout: {{ settingsService.setting.pointsShutouts }} points
  • Points in the Stanley Cup Final are doubled
  • Points are updated on this website in real time every 10 minutes

  • Tiebreakers:
    1. Total Points
    2. Total Points from Centers
    3. Total Points from Wingers
    4. Total Points from Defensemen
    5. Total Points from Goalies
    6. Total Overall Goals
    7. Split Money
